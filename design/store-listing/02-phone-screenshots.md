# Phone Screenshots — 1080 × 1920 px (9:16)

Generate 8 screenshots (Play allows up to 8; use all 8 — more real estate to
sell the app). Each prompt below is a self-contained "marketing screenshot":
a phone mockup showing a real app screen, plus a short headline caption and
brand-colored background. Keep the caption style **identical** across all 8
so the set reads as one coherent set on the store listing carousel.

**Shared style block** — prepend this to every prompt if your tool doesn't
retain context between generations:

```
Style: modern mobile app store screenshot, flat design, no photorealism.
Background is a solid or very subtle gradient background in brand color
(specified per screen below). A single Android phone mockup (simple rounded-
rectangle silhouette, thin dark bezel, no realistic reflections) is centered
in the lower 80% of the frame, tilted 0 degrees (straight-on), displaying
the described app screen at native mobile resolution with crisp, legible
UI text. A short bold marketing headline sits above the phone in white or
dark text (contrast-appropriate to the background), Inter/SF Pro Display
Bold, roughly 56-64px, max 2 lines. Brand colors: blue #1A6BB5, orange
#F05A1A, green #6DC400, dark text #1A1A1A, light background #F8F9FA, card
white #FFFFFF with 16px rounded corners and soft shadow. Output exactly
1080x1920 pixels, portrait.
```
